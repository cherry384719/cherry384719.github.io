## 1. [**api.ygking.top**](https://api.ygking.top/)

**纯免费**，基于 Cloudflare Pages + D1 数据库的QQ音乐 API 服务，访问网站可查看接口使用方法。


- 来自[L站帖子](https://linux.do/t/topic/1553601)
- 相关[github项目](https://github.com/tooplick/qq-music-api)



## 2. [**TuneHub音乐API**](https://tunehub.sayqz.com/)

需要用 LinuxDO[^1]账号登录，签到 或 LDC[^2] 兑换得积分

- 来自[L站帖子](https://linux.do/t/topic/1509257)


## 3. [**GD音乐台(music.gdstudio.xyz)**](https://music-api.gdstudio.xyz/api.php)

开放免费接口，限制访问频率（5分钟内不超50次请求）。


## 4. [**【DownList】网易云音乐下载公益站**](https://downlist.shfu.cn/)


- 来自[L站帖子](https://linux.do/t/topic/1577173)

食用方法：
- 打开链接，用 LinuxDo 登录
- 点击升级配置 （确保 购买会员 和 自备cookie 二选一完成）
